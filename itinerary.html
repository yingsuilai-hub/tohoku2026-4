每日行程總覽

實際車次請以 JR / 巴士官方最終時刻為準。

D1｜7/21（二）桃園 → 青森

BR122 抵達青森後入住 ReLabo。晚間以青森站周邊、A-FACTORY、港邊散步為主。

D2｜7/22（三）青森市區 / 弘前彈性日

WA RASSE、A-FACTORY、青森美術館、三內丸山；若天氣佳可改弘前城與蘋果派。

D3｜7/23（四）奧入瀨溪流

08:10 青森站搭 JR BUS 湖2號 → 10:40 子之口下車；沿溪流下坡走到石戶／溪流館。回程建議：石戶 14:19 或溪流館 14:27 → 青森 16:29；備案 15:34／15:42 → 17:44。

D4｜7/24（五）TOHOKU EMOTION

青森/八戶 → 久慈，14:18 搭甜點列車到八戶。

D5｜7/25（六）移動仙台

八戶或青森搭新幹線到仙台，與後到家人會合。

D6｜7/26（日）松島

日本三景，搭遊船、瑞巖寺、五大堂、海鮮午餐。

D7｜7/27（一）銀山溫泉 / 山形

重點放溫泉街、足湯、午餐；避免排太多景點。

D8｜7/28（二）秋保溫泉 / 仙台近郊

適合長輩泡湯與午餐，避免長距離移動。

D9｜7/29（三）仙台採買 / 備案日

牛舌、百貨、伴手禮，作為雨天或體力緩衝。

D10｜7/30（四）仙台 → 桃園

早上最後採買，前往仙台機場搭 BR117 返台。
